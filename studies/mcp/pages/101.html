圖 A · 設計與使用
三個角色,一條專屬線

把它想成「插座」:Host 是你的房子,每個 Server 是一個電器,Client 是那條專屬的電源線—— 一個電器一條線,不是所有電器共用一條延長線。點下面場景裡的房子、電線、電器,看對應哪個角色。

六個 primitive,清楚分兩邊——這條線就是「誰對什麼有權威」的邊界。多數人以為 MCP 是單向 (server 給,client 拿),右邊三個其實是反過來:server 也能敲 client 的門。
